## IntelliJ를 시작하시는 분들을 위한 IntelliJ 가이드 - 인프런 향로
- - -
**실행환경**
- 현재 포커스 실행: Ctrl + Shift + R
- 이전 실행: Shift + F10

**탭**
- 탭 닫기: Ctrl + F4
- 탭 이동: Alt + 방향

**라인**
- 라인 복제: Ctrl + D
- 라인 삭제: Ctrl + X
- 라인 합치기: Ctrl + Shift + J
- 라인 옮기기(구문내): Alt + Shift + 방향
- 라인 옮기기(구문외): Ctrl + Shift + 방향
- Element 옮기기: Ctrl + Shift + Alt + 방향
